# Secure Authentication API

## Tech Stack
- .NET Web API
- Clean Architecture
- CQRS + MediatR
- JWT Authentication
- Refresh Tokens
- Role & Policy Authorization

## Features
- User Registration
- Secure Login
- JWT Access Token
- Refresh Token Flow
- Role-Based Authorization
- Policy-Based Authorization

## How to Run
1. Configure JWT settings in appsettings.json
2. Apply database migrations
3. Run the API
4. Use Postman / Swagger to test

## Security
- Password hashing + salting
- Token expiration
- Refresh token rotation
